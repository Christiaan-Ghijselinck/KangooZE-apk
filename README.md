# Kangoo ZE 2014

The KangooZE.apk is an Android application file that enables diagnostic and monitoring of the Renault electric vehicle Z.E. (Zero Emission) like the Kangoo ZE, and Fluence ZE.  It works via an ELM327 compatible bluetooth dongle.

The KangooZE.apk file is freely available and can be downloaded via the <>Code button above.
